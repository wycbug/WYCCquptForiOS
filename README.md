## 功能：
* 登陆界面出入账号后能判断登陆成功没有，弹出提示框
* 登陆成功后会把获取的用户信息存到数据库里（如果数据库里没有数据的话）
* 单独写的一个getClassBookData，能解析出每周的数据按照星期存储
* 尽量用了MVC
## 下一步实现的功能
